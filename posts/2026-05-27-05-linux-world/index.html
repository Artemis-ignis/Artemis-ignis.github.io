2026-05-27 강의자료 05: 리눅스의 세계로

리눅스는 서버 환경에서 많이 쓰이는 운영체제다. 개인 컴퓨터에서는 윈도우를 많이 쓰지만, 웹서비스를 배포하고 운영하는 서버에서는 리눅스를 자주 만난다.

리눅스를 배우는 이유

리눅스는 안정적이고 비용 부담이 낮으며, 서버 자동화 도구와 잘 연결된다. 클라우드 서버를 만들면 기본 환경이 리눅스인 경우도 많다.

개발자가 로그를 확인하거나, 파일을 수정하거나, 서버 프로세스를 재시작할 때 리눅스 명령어를 사용한다.

CLI 환경

CLI는 글자로 명령을 입력하는 환경이다. 처음에는 버튼이 없어서 어렵게 느껴지지만, 반복 작업을 자동화하기 좋다.

예를 들어 ls로 파일 목록을 보고, cd로 디렉터리를 이동하고, mkdir로 폴더를 만든다. 이런 명령어가 익숙해지면 서버에서 직접 문제를 확인할 수 있다.

오늘 가져갈 것

리눅스는 개발자가 서버와 대화하기 위한 기본 언어에 가깝다. 윈도우 사용자라도 WSL Ubuntu 같은 환경을 통해 리눅스 명령어에 익숙해지는 것이 좋다.
