2026-05-28 강의자료 01: 네트워크 개요

네트워크는 컴퓨터들이 서로 데이터를 주고받을 수 있게 연결된 구조다. 웹사이트 접속, 메신저, 이메일, 파일 다운로드는 모두 네트워크 위에서 움직인다.

IP 주소

IP 주소는 네트워크에서 장치를 찾기 위한 주소다. 집 주소가 있어야 택배가 도착하듯이, 컴퓨터도 IP 주소가 있어야 데이터를 주고받을 수 있다.

도메인 이름은 사람이 읽기 쉽게 만든 이름이고, 실제 통신에서는 DNS를 통해 IP 주소로 바뀐다.

통신 방식

네트워크 통신은 데이터를 작은 단위로 나누어 보내고, 상대방이 다시 조립하는 방식으로 이해할 수 있다. 데이터가 한 번에 통째로 이동하는 것이 아니라 여러 조각이 규칙에 따라 움직인다.

OSI 7계층

OSI 7계층은 네트워크 통신을 단계별로 이해하기 위한 모델이다. 실제 개발에서 매번 7계층을 모두 외우기보다, 문제가 어느 층에서 생겼는지 좁히는 도구로 보면 좋다.

오늘 가져갈 것

네트워크는 웹서비스의 도로망이다. 클라이언트와 서버가 요청과 응답을 주고받으려면 주소, 경로, 통신 규칙이 모두 맞아야 한다.
