2026-05-27 강의자료 06: 리눅스 파일 시스템

리눅스 파일 시스템은 /를 기준으로 시작한다. 윈도우의 C 드라이브처럼 여러 드라이브가 먼저 보이는 구조와 다르게, 리눅스는 하나의 루트 아래에 파일과 디렉터리가 정리된다.

주요 디렉터리

경로 의미
/home 사용자 개인 파일
/etc 설정 파일
/var 로그와 변하는 데이터
/bin 기본 실행 파일
/usr 사용자 프로그램과 라이브러리
/tmp 임시 파일

서버에서 설정을 바꿀 때 /etc, 로그를 볼 때 /var/log, 사용자 작업 파일을 찾을 때 /home을 자주 만난다.

절대 경로와 상대 경로

절대 경로는 루트부터 시작하는 전체 주소다. 상대 경로는 현재 위치를 기준으로 한 주소다.

명령어를 실행할 때 현재 위치가 어디인지 모르면 파일을 찾지 못하는 일이 생긴다. 그래서 pwd로 현재 위치를 확인하는 습관이 필요하다.

오늘 가져갈 것

리눅스 파일 시스템은 서버 작업의 지도다. 어떤 파일이 어디에 있어야 하는지 알면 설정, 로그 확인, 배포 작업을 훨씬 안정적으로 할 수 있다.
