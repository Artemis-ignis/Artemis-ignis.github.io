2026-05-29 강의자료 01: 기초 자료형 - Python으로의 초대

이 자료는 파이썬 문법의 출발점이다. 코드를 처음 배울 때 가장 먼저 익혀야 할 것은 값을 출력하고, 변수에 저장하고, 자료형을 구분하는 일이다.

출력과 변수

print()는 값을 화면에 보여주는 함수다. 변수는 값을 담는 이름표다.

name = "사용자 A"
age = 20

print(name)
print(age)

변수 이름은 의미 있게 짓는 것이 좋다. x보다 user_name, total_price처럼 읽히는 이름이 나중에 코드를 이해하기 쉽다.

자료형

자료형은 값의 종류다. 정수, 실수, 문자열, 참/거짓 같은 값은 서로 다른 방식으로 다뤄진다.

자료형 예시
int 10
float 3.14
str "hello"
bool True

숫자 10과 문자열 "10"은 다르다. 계산을 하려면 숫자 자료형이어야 한다.

오늘 가져갈 것

파이썬 기초는 변수와 자료형에서 시작한다. 값을 어떤 이름으로 저장하고, 그 값이 어떤 종류인지 이해하면 이후 조건문과 리스트도 훨씬 쉽게 이어진다.
