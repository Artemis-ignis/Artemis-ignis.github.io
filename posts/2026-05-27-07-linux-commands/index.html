2026-05-27 강의자료 07: 알고 쓰는 리눅스 명령어

리눅스 명령어는 단순히 외워야 하는 단어가 아니라 서버와 대화하는 방법이다. 명령어를 알면 파일을 찾고, 내용을 확인하고, 로그에서 필요한 줄을 빠르게 골라낼 수 있다.

기본 명령어

명령어 의미
pwd 현재 위치 확인
ls 목록 보기
cd 디렉터리 이동
cp 복사
mv 이동 또는 이름 변경
rm 삭제
cat 파일 내용 보기
grep 문자열 검색

문자열 검색

로그 파일이 길 때 처음부터 끝까지 읽는 것은 비효율적이다. grep을 쓰면 원하는 단어가 포함된 줄만 찾을 수 있다.

정규표현식은 문자열 패턴을 표현하는 방법이다. 이메일 형식, 숫자 패턴, 특정 접두어를 가진 단어를 찾을 때 사용할 수 있다.

오늘 가져갈 것

CLI는 익숙해질수록 강력하다. 마우스로 반복하는 일을 명령어로 정리하면 나중에 스크립트로 자동화할 수 있다. 개발자는 명령어를 통해 서버 상태를 빠르게 읽을 수 있어야 한다.
