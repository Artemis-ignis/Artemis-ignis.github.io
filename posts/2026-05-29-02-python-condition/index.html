2026-05-29 강의자료 02: 조건문 - 진실 혹은 거짓

조건문은 상황에 따라 다른 코드를 실행하게 해준다. 컴퓨터에게 “이 조건이면 이렇게 하고, 아니면 저렇게 해"라고 알려주는 문법이다.

입력받기

input()은 사용자에게 값을 입력받는다. 입력받은 값은 기본적으로 문자열이기 때문에 숫자 계산을 하려면 자료형을 바꿔야 한다.

age = int(input("나이를 입력하세요: "))

if 조건문

조건문은 비교 결과가 참인지 거짓인지에 따라 실행 흐름을 나눈다.

score = 85

if score >= 80:
    print("통과")
else:
    print("다시 도전")

파이썬에서 들여쓰기는 문법이다. 들여쓰기가 틀리면 코드의 의미가 달라지거나 오류가 난다.

비교 연산

==, !=, >, <, >=, <= 같은 비교 연산자는 조건을 만들 때 사용한다.

오늘 가져갈 것

조건문을 배우면 프로그램이 한 방향으로만 흐르지 않고 상황에 맞게 판단할 수 있다. 로그인 성공/실패, 점수 판정, 권한 확인 같은 기능의 기본이 된다.
