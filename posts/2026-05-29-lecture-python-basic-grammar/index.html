2026-05-29 이어드림스쿨 강의자료 정리: 파이썬 기본 문법

이 글은 2026-05-29 파이썬 기본 문법 강의자료를 기준으로 정리한 학습 기록이다. 앞선 수업에서 컴퓨터, 서버, 네트워크를 봤다면, 이날은 직접 코드를 쓰기 위한 기본 문법을 다뤘다.

강의자료 순서

순서 강의자료 핵심 주제
1 기초 자료형: Python으로의 초대 출력, 변수, 자료형, 연산, 문자열
2 조건문: 진실 혹은 거짓 입력, 비교, 조건 분기
3 리스트: 모아모아 다모아 리스트 활용, 여러 값 저장

출력과 변수

파이썬에서 print()는 화면에 값을 보여주는 가장 기본적인 명령이다.

print("Hello Python")

변수는 값을 담아두는 이름표다. 숫자나 문자열을 변수에 저장해두면 나중에 다시 사용할 수 있다.

name = "사용자 A"
age = 20

print(name)
print(age)

변수 이름은 사람이 읽기 쉽게 짓는 것이 중요하다. a, b보다 user_name, total_price처럼 의미가 보이는 이름이 나중에 코드를 읽기 쉽다.

자료형과 연산

자료형은 값의 종류다. 파이썬은 숫자, 문자열, 참/거짓, 리스트 같은 여러 자료형을 다룬다.

자료형 예시 의미
int 10 정수
float 3.14 실수
str "hello" 문자열
bool True, False 참/거짓
list [1, 2, 3] 여러 값을 담는 묶음

자료형을 이해해야 오류를 줄일 수 있다. 숫자 10과 문자열 "10"은 화면에는 비슷해 보여도 파이썬에서는 다른 값이다.

문자열과 입력

문자열은 글자 데이터다. 이름, 이메일, 주소, 문장처럼 글자로 된 값은 문자열로 다룬다.

message = "오늘은 파이썬을 배웠다"
print(message)

input()은 사용자에게 값을 입력받을 때 쓴다.

name = input("이름을 입력하세요: ")
print(name)

다만 input()으로 받은 값은 기본적으로 문자열이다. 숫자 계산을 하려면 int()나 float()로 바꿔야 한다.

조건문

조건문은 상황에 따라 다른 코드를 실행하게 해준다. “비가 오면 우산을 챙긴다” 같은 판단을 코드로 쓰는 것이다.

score = 85

if score >= 80:
    print("통과")
else:
    print("다시 도전")

조건문에서 중요한 것은 들여쓰기다. 파이썬은 중괄호 대신 들여쓰기로 코드 블록을 구분한다.

리스트

리스트는 여러 값을 한 번에 담는 자료형이다. 학생 이름, 상품 목록, 점수 목록처럼 비슷한 성격의 값을 묶을 때 사용한다.

scores = [90, 80, 70]

print(scores[0])
print(scores[1])

리스트의 첫 번째 위치는 1이 아니라 0이다. 그래서 scores[0]은 첫 번째 값인 90을 의미한다.

리스트는 값을 추가하거나 반복해서 처리할 때 특히 유용하다.

names = ["사용자 A", "사용자 B", "사용자 C"]

for name in names:
    print(name)

오늘의 정리

파이썬 기본 문법은 코딩의 문장 부호와 같다. 출력, 변수, 자료형, 조건문, 리스트를 알아야 더 복잡한 데이터 분석이나 웹 개발 코드로 넘어갈 수 있다. 처음에는 문법을 외우는 느낌이지만, 작은 예제를 반복하면 “컴퓨터에게 순서대로 설명하는 법"에 가까워진다.
