2026-05-26 강의자료 06: 검색엔진 최적화 SEO

SEO는 검색엔진 최적화다. 좋은 글을 쓰는 것만으로는 부족하고, 검색엔진이 그 글을 이해하기 쉽게 구조를 정리해야 한다. SEO가 필요한 이유 웹사이트는 사용자가 직접 주소를 입력해서 들어오기도 하지만, 검색을 통해 발견되는 경우가 많다. 검색엔진은 페이지 제목, 설명, 본문 구조, 링크, 속도 등을 참고해 어떤 페이지인지 판단한다. 기본 요소 제목은 페이지의 주제를 분명하게 보여줘야 한다. 설명은 검색 결과에서 사용자가 클릭할지 판단하는 짧은 요약문 역할을 한다. Hugo 글에서 front matter에 title, description, tags, categories를 넣는 것도 같은 이유다. 사람과 검색엔진 모두에게 글의 성격을 알려준다.
