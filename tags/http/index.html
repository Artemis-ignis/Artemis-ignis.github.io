2026-05-28 강의자료 02: 인터넷과 HTTP

HTTP는 웹에서 클라이언트와 서버가 대화하는 규칙이다. 브라우저가 서버에 요청을 보내고, 서버가 상태 코드와 데이터를 응답으로 돌려준다. HTTP 요청과 응답 사용자가 웹페이지에 접속하면 브라우저는 서버에 요청을 보낸다. 서버는 요청을 처리한 뒤 HTML, CSS, 이미지, JSON 같은 데이터를 돌려준다. 요청에는 주소, 메서드, 헤더, 필요한 데이터가 포함될 수 있다. 응답에는 상태 코드와 실제 데이터가 들어온다. 상태 코드 코드 의미 200 성공 301/302 이동 400 잘못된 요청 401 인증 필요 403 권한 없음 404 찾을 수 없음 500 서버 오류 오류 코드를 읽을 수 있으면 문제 위치를 빠르게 좁힐 수 있다.
