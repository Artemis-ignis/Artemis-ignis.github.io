2026-05-26 강의자료 02: 앱·웹 서비스를 위한 모든 것

이 글은 [수업자료] 앱_웹 서비스를 위한 모든 것.pdf를 보고 다시 정리한 복습 기록이다. 슬라이드를 그대로 옮기기보다는, 나중에 봤을 때 “아, 이게 그 말이었지” 하고 떠올릴 수 있게 쉬운 말과 예시 중심으로 적었다. 한 줄 요약 웹서비스는 예쁜 화면 하나가 아니라, 사용자 화면과 서버 처리, 데이터 저장, 관리자 운영, 보안이 같이 움직이는 시스템이다. 수업 흐름 서비스를 기획할 때 초보자는 보통 ‘사용자가 보는 화면’부터 떠올린다. 하지만 실제 서비스는 사용자가 버튼을 누른 뒤 서버에서 무엇을 확인하고, 어떤 데이터를 저장하고, 관리자는 어디서 검수하고, 문제가 생기면 어떻게 복구할지까지 생각해야 한다. 화면은 입구이고, 뒤쪽에는 백엔드 로직과 데이터베이스, 인증, 권한, 로그, 분석이 붙는다.
