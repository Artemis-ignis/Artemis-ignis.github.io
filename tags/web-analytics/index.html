2026-05-26 강의자료 06: 검색엔진 최적화 SEO

이 글은 [수업자료] 검색엔진 최적화 SEO.pdf를 보고 다시 정리한 복습 기록이다. 슬라이드를 그대로 옮기기보다는, 나중에 봤을 때 “아, 이게 그 말이었지” 하고 떠올릴 수 있게 쉬운 말과 예시 중심으로 적었다. 한 줄 요약 SEO는 검색엔진이 내 페이지를 잘 찾고, 잘 이해하고, 사용자에게 보여 줄 만하다고 판단하게 만드는 기본 정리 작업이다. 수업 흐름 검색엔진은 웹페이지를 돌아다니며 수집하고, 내용을 색인하고, 검색어와 맞는 페이지를 순서대로 보여 준다. 그래서 제목, 설명, URL, 본문 구조, 링크, 페이지 속도 같은 요소가 중요하다. SEO는 꼼수를 쓰는 기술이라기보다, 사람과 검색엔진이 모두 이해하기 좋게 웹페이지를 정리하는 일에 가깝다.

2026-05-26 이어드림스쿨 수업 정리: 데이터 분석은 그래프보다 설명이다

오늘 수업에서 데이터 분석 파트는 코딩보다 “설명"에 가까웠다. 표를 만들고 그래프를 그리는 건 분석의 끝이 아니라 중간 과정이다. 진짜 중요한 건 그 숫자가 무슨 뜻인지 말할 수 있는가다. 데이터 분석이란 무엇인가 데이터 분석은 데이터를 보고 의미를 찾아내고, 그 의미를 다른 사람이 이해할 수 있게 설명하는 일이다. 방문자 수 그래프를 만들었다고 분석이 끝나는 건 아니다. 방문자가 왜 줄었는지, 어느 페이지에서 많이 나갔는지, 다음에는 무엇을 바꾸면 좋을지를 말할 수 있어야 한다.
