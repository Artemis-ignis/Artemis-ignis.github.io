2026-05-26 이어드림스쿨 수업 정리: 공공데이터 OpenAPI 읽는 법

아파트 매매 실거래가 자료 기술문서는 오늘 배운 API 개념을 실제 문서에서 확인하는 예시로 보기 좋았다. 수업에서 배운 REST API, 요청과 응답, XML, 인증키가 한꺼번에 나온다. 중요한 점부터 정리하면, 공공데이터 API는 주소만 브라우저에 넣는다고 끝나는 게 아니다. 필요한 파라미터를 같이 보내야 하고, 인증키도 안전하게 다뤄야 한다. OpenAPI는 공개된 약속이다 OpenAPI는 외부에서 사용할 수 있도록 공개된 API다. 공공데이터 API는 정부나 기관이 가진 데이터를 정해진 방식으로 요청할 수 있게 해준다.
