2026-05-28 강의자료 02: 인터넷과 HTTP

이 글은 [수업 자료] 인터넷과 HTTP.pdf를 보고 다시 정리한 복습 기록이다. 슬라이드를 그대로 옮기기보다는, 나중에 봤을 때 “아, 이게 그 말이었지” 하고 떠올릴 수 있게 쉬운 말과 예시 중심으로 적었다. 한 줄 요약 HTTP는 웹브라우저와 서버가 요청과 응답으로 대화하기 위한 대표적인 약속이다. 수업 흐름 웹은 요청과 응답의 반복으로 움직인다. 사용자가 링크를 누르거나 버튼을 누르면 브라우저가 서버에 HTTP 요청을 보내고, 서버는 HTML, JSON, 이미지 같은 응답을 돌려준다. 상태 코드는 요청 결과를 숫자로 알려 준다. 200번대는 성공, 300번대는 이동, 400번대는 클라이언트 쪽 문제, 500번대는 서버 쪽 문제로 크게 볼 수 있다.
