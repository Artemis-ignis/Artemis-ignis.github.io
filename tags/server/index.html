2026-05-27 강의자료 08: 프로세스와 네트워크

2026-05-27 강의자료 05: 리눅스의 세계로

이 글은 [강의자료] 리눅스의 세계로.pdf를 보고 다시 정리한 복습 기록이다. 슬라이드를 그대로 옮기기보다는, 나중에 봤을 때 “아, 이게 그 말이었지” 하고 떠올릴 수 있게 쉬운 말과 예시 중심으로 적었다. 한 줄 요약 리눅스는 서버와 개발 자동화에서 많이 쓰이는 운영체제라서, 웹서비스를 배포하고 운영하려면 기본 사용법을 익혀야 한다. 수업 흐름 개인 컴퓨터에서는 Windows를 많이 쓰지만, 실제 서버 환경은 리눅스 기반인 경우가 많다. 비용이 낮고, 안정적으로 오래 켜 두기 좋고, 자동화 도구와 배포 도구가 리눅스 중심으로 돌아가는 경우가 많기 때문이다. Windows 사용자도 WSL Ubuntu를 쓰면 리눅스 환경을 연습할 수 있다.

2026-05-26 이어드림스쿨 수업 정리: 앱·웹 서비스는 화면 뒤에서 어떻게 돌아갈까

앱_웹 서비스를 위한 모든 것 자료에서 핵심은 하나였다. 서비스는 예쁜 화면만으로 완성되지 않는다. 사용자가 보는 화면 뒤에는 서버, 데이터베이스, 관리자 기능, 보안, 개인정보 처리까지 줄줄이 붙어 있다. 초보자는 화면을 먼저 본다. 버튼이 있고, 입력창이 있고, 결과가 보이면 서비스처럼 느껴진다. 하지만 실제 운영 가능한 서비스는 버튼을 누른 뒤에 무슨 일이 일어나는지가 더 중요하다. 클라이언트와 서버 클라이언트는 사용자가 직접 보는 쪽이다. 웹브라우저, 스마트폰 앱, 화면에 보이는 버튼과 입력창이 여기에 가깝다.
